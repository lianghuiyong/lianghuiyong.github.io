## 一. 主题
1、基本主题
```
    <style name="Base.AppTheme" parent="Theme.AppCompat.Light.NoActionBar">
        <!--text style-->
        <item name="android:textColor">@color/textColor</item>
        <item name="colorAccent">@color/colorAccent</item>
        <item name="android:windowAnimationStyle">@style/AnimationActivity</item>
    </style>
```
<!--more-->
2、5.0以上系统适配主题和5.0以下的主题
```
    <!--标题栏透明-->
    <style name="Base.TransParent" parent="Base.AppTheme">
        <item name="android:windowDrawsSystemBarBackgrounds">true</item>
        <item name="android:statusBarColor">@android:color/transparent</item>
    </style>
```

```
    <!--标题栏透明-->
    <style name="Base.TransParent" parent="Base.AppTheme">
    </style>
```
3、页面

```
    <style name="TransParentTheme" parent="Base.TransParent">
        <item name="colorPrimary">@color/mainColorPrimary</item>
        <item name="colorPrimaryDark">@color/mainColorPrimaryDark</item>
    </style>
```
## 二. 页面使用
```
<android.support.v4.widget.DrawerLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    android:id="@+id/drawer_layout"
    android:layout_width="match_parent"
    android:layout_height="match_parent">
    <!--其他布局-->
    <......>
    <!--侧栏引用布局-->
    <include
        layout="@layout/nav_header_main"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:layout_gravity="start"/>
</android.support.v4.widget.DrawerLayout>
```
